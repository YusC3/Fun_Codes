Tic-Tac-Toe v1.0
- - - - - - - - -
DISCLAIMER: This game is intended to be run from the command line. 
It also includes custom library gameMake for text based game creation

Rules:
Two players compete to get 3 symbols to match in a row, column
or diagnolly. If 3 symbols match in a given vector, a player has won 
and the game ends. If after 9 turns no one has won, the game ends in a 
draw. Players are only allowed to enter in numbers that correspond to
an open position (i.e. no alphabet characters or numbers that arent a
position on the board or an empty position).

To Run:
Enter the directory that contains the files, type:
	python play_game.py
